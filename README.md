# What To Code CLI
Simple cli applicatoin that uses the what-to-code api

**How to install -**

```bash
git clone https://github.com/maytees/whattocode.git

cd whattocode

cargo build --release

sudo cp ./target/release/whattocode /usr/bin/whattocode
```
